# Awan Cloud Canvas

A modern, scalable web application for cloud transformation and AI services. Built with React, TypeScript, Tailwind CSS, and advanced 3D animations.

## Features

- **Modern UI**: Built with React 18 and ShadCN/UI components
- **3D Animations**: Particle effects and advanced animations with Three.js and GSAP
- **Responsive Design**: Fully responsive across all devices
- **Dark Mode**: Built-in dark mode support
- **Type Safe**: Full TypeScript support
- **Testing**: Unit tests with Vitest and E2E tests with Playwright
- **Performance**: Optimized builds with Vite and SWC

## Quick Start

```bash
# Install dependencies
bun install  # or npm install

# Start development server
bun run dev

# Build for production
bun run build

# Run tests
bun run test
```
